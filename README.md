# 小派 (Pi-Robot) - 智能机器人项目

## 项目概述

小派是一个基于树莓派的智能机器人项目，采用模块化异步驱动架构，支持语音交互、远程控制和自主导航。

---

## 已开发功能

### 1. 核心状态机

| 状态 | 值 | 描述 | 互斥规则 |
|------|-----|------|----------|
| IDLE | `"idle"` | 待机状态 | 可被语音唤醒或远程接入 |
| AWAKE | `"awake"` | 语音唤醒状态 | 拒绝远程接入 |
| REMOTE | `"remote"` | 远程控制状态 | 屏蔽唤醒词 |

**状态流转规则：**
- IDLE → AWAKE：检测到唤醒词
- IDLE → REMOTE：远程登录成功
- AWAKE → IDLE：超时无操作或检测到退出词
- REMOTE → IDLE：远程断开或退出

### 2. 语音交互模块

| 功能 | 描述 | 状态 |
|------|------|------|
| 离线唤醒 | Sherpa-ONNX 流式唤醒词检测 | ✅ 已实现 |
| 语音识别 | DashScope ASR 实时语音转文字 | ✅ 已实现 |
| 语音合成 | DashScope TTS 文字转语音（流式） | ✅ 已实现 |
| LLM对话 | DeepSeek/Qwen 大模型集成 | ✅ 已实现 |
| 打断机制 | 用户说话时自动打断TTS播报 | ✅ 已实现 |
| 超时退出 | AWAKE状态超时自动返回IDLE | ✅ 已实现 |
| 问候语 | IDLE→AWAKE时随机播放问候语 | ✅ 已实现 |
| 告别语 | AWAKE→IDLE时随机播放告别语 | ✅ 已实现 |

### 3. 插件系统

| 插件 | 功能 | 状态 |
|------|------|------|
| LED控制 | 状态指示、呼吸灯效果 | ✅ 已实现 |
| 电机控制 | 前进/后退/左转/右转/停止 | ✅ 已实现 |
| 摄像头 | 视频捕获基础功能 | ✅ 已实现 |
| 远程控制 | WebSocket通信框架 | ✅ 已实现 |
| 超声波 | 距离检测 | ✅ 已实现 |
| 蜂鸣器 | 声音提示 | ✅ 已实现 |

### 4. 意图识别

| 意图 | 关键词 | 描述 |
|------|--------|------|
| MOVE_FORWARD | 前进、向前走、往前走 | 向前移动 |
| MOVE_BACKWARD | 后退、向后退、倒着走 | 向后移动 |
| TURN_LEFT | 左转、向左转 | 向左转动 |
| TURN_RIGHT | 右转、向右转 | 向右转动 |
| STOP | 停止、停、住手 | 紧急停止 |
| WEATHER_QUERY | 天气、天气预报 | 查询天气 |
| EXIT | 退出、再见、拜拜 | 返回IDLE状态 |

## 核心组件职责

| 组件 | 职责 | 关键方法 |
|------|------|----------|
| **Brain** | 状态机核心，管理状态流转 | `think()`, `transition_to()` |
| **IntentHandler** | 意图识别处理器 | `match_intent(text)` |
| **CommandQueue** | 命令队列管理 | `enqueue()`, `execute()` |
| **MemoryManager** | 对话历史管理 | `save_memory()`, `load_memory()` |

## 常见问题

### Q: 摄像头无法初始化？

A: 需要修改虚拟环境配置，使其包含系统站点包：
```bash
nano venv/pyvenv.cfg
# 将 include-system-site-packages = false 改为 true
```

### Q: TTS没有声音？

A: 检查环境变量 `DASHSCOPE_API_KEY` 是否正确配置。

### Q: 如何添加新意图？

A: 在 `src/core/intent_keywords.py` 中添加关键词，在 `intent_mapping.py` 中配置映射关系。
