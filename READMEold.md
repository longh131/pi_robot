小派 (Pi-Robot) 开发文档 v1.0.0
版本状态： 稳定版 (RC-1)
核心特性： 低功耗架构、离线唤醒、大模型对话、异步语音合成、娱乐控制

1. 项目架构概览
小派采用 模块化异步驱动架构，确保在树莓派有限的硬件资源下，能够同时处理语音采集、语义分析与多模态输出。

核心模块：
- Brain (核心)：控制中心，负责状态机管理（待机/唤醒/侦听/思考/播报）与任务调度
- Wake (感知)：基于 Sherpa-ONNX 的流式离线唤醒
- ASR (听觉)：语音转文字引擎（基于 Whisper）
- LLM (思维)：连接大语言模型，处理对话与意图
- TTS (表达)：语音合成输出（基于 Qwen3-TTS）
- Parser (协议)：解析 LLM 返回的内容，拆分语音与指令

插件系统：
- LED 插件：状态指示灯控制
- 电机插件：机器人运动控制（前进、后退、左转、右转、停止）
- 娱乐插件：摄像头（拍照、录像）和音乐播放功能
- 跟随插件：自动跟随功能
- 跳舞插件：预设舞蹈动作

2. 关键技术突破 (v1.0.0 核心)

2.1 CPU 占用率优化
针对早期版本 CPU 占用率飙升至 120%+ 导致死机的问题，v1.0.0 引入了 "呼吸循环" (Breathing Loop) 机制：
- 唤醒端限速：在 WakeEngine 的主循环中强制加入 time.sleep(0.01)，将轮询频率限制在 100Hz 左右，成功将 CPU 占用从 120% 降至 15%~25%
- 非阻塞回调：采用 sounddevice 异步回调捕获音频，确保在休眠期间不会丢失任何语音片段

2.2 鲁棒性改进
- 日志探针 (Probes)：内置时间增量分析（Delta-Time Analysis），可实时监测各模块执行耗时
- 线程守护：对话逻辑在独立守护线程运行，支持 cancel_event 紧急打断

2.3 意图识别优化
- 本地指令优先：常用控制指令（拍照、录像、音乐播放等）直接本地识别执行，无需调用 LLM
- 紧急停止词：支持配置全局紧急停止词，可立即停止所有活动

3. 已实现功能清单

3.1 语音交互
- 离线唤醒：支持自定义唤醒词（如"小派同学"、"小派"、"派派"等）
- 流式识别：实时将用户语音转化为文本
- 智能理解：集成 LLM，支持角色设定（专属机器人助手）
- 打断机制：在机器人说话过程中，检测到新语音可立即停止当前动作
- 超时退出：唤醒后无操作超过设定时间自动进入待机状态

3.2 娱乐控制
- 拍照功能：语音指令"拍照"、"照相"、"拍张照片"触发
- 录像功能：语音指令"录像"、"开始录像"开始录制，"停止录像"、"结束录像"停止录制
- 音乐播放：语音指令"播放音乐"、"放歌"播放音乐，支持暂停、停止、上一首、下一首

3.3 运动控制
- 前进/后退：语音指令控制机器人移动
- 左转/右转：语音指令控制机器人转向
- 停止：语音指令"停止移动"、"停止走动"停止电机

3.4 状态管理
- 待机状态 (IDLE)：等待唤醒词或退出
- 唤醒状态 (AWAKENING)：播放欢迎语，进入侦听
- 侦听状态 (LISTENING)：等待用户语音输入
- 思考状态 (THINKING)：处理用户请求
- 播报状态 (SPEAKING)：播放语音回复

4. 配置文件说明

4.1 机器人配置 (data/robot_profile.json)
- wake_words：唤醒词列表
- exit_words：退出词列表（如"退出"、"再见"）
- emergency_stop_words：紧急停止词列表（如"停止"、"闭嘴"）
- greeting：欢迎语列表（随机选择播放）
- farewell：告别语列表（随机选择播放）

4.2 意图关键词配置 (config/intent_keywords.py)
- WAKE_WORDS：唤醒词关键词
- EXIT_WORDS：退出词关键词
- EMERGENCY_STOP_WORDS：紧急停止词关键词
- MOVEMENT_KEYWORDS：运动控制关键词
- ENTERTAINMENT_KEYWORDS：娱乐控制关键词（拍照、录像、音乐）

5. 硬件接入参考

5.1 LED 状态灯
- 引脚：GPIO 17 (Physical Pin 11)
- 逻辑：根据机器人状态自动控制

5.2 摄像头
- 支持：Raspberry Pi Camera Module
- 分辨率：拍照 2592x1944，录像 1280x720
- 存储：照片保存在 data/media/photo_*.jpg，录像保存在 data/media/video_*.h264

5.3 电机驱动
- 支持：L298N 电机驱动板
- 引脚：GPIO 12, 13, 18, 19

6. 快速开始

6.1 环境准备
```bash
# 安装系统依赖
sudo apt install libcamera-tools python3-libcamera

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装 Python 依赖
pip install -r requirements.txt
```

6.2 启动机器人
```bash
source venv/bin/activate
python src/core/brain.py
```
